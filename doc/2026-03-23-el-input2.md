# ElInput2 组件说明

## 目的

作为 `el-input` 的全局 drop-in replacement，统一为项目中所有输入框添加默认 `maxlength=4000`，防止超长字符串提交。

通过 `main.js` 全局覆盖注册：

```js
Vue.use(ElementUI)
Vue.component('ElInput', ElInput2)
```

## 用法

与原 `el-input` 完全兼容，额外支持以下行为：

| 场景 | 写法 |
|------|------|
| 使用默认限制（4000） | `<el-input v-model="val" />` |
| 覆盖为自定义限制 | `<el-input v-model="val" :maxlength="100" />` |
| 禁用限制（无上限） | `<el-input v-model="val" :maxlength="null" />` |

## 设计细节

- `inheritAttrs: false` + `v-bind="mergedAttrs"`：将所有透传 attr 与 `maxlength` 合并后传给内层 `element-input`
- `maxlength` 声明为 prop（非 attr），不会出现在 `$attrs` 中，避免冲突
- `v-on="listeners"` 转发所有事件，但 `input` 事件单独处理以支持 `v-model`
- `$scopedSlots` 转发所有 slot（Vue 2.6+ 中已包含非 scoped slot，无需再遍历 `$slots`）
- `mounted` 里通过 `Object.defineProperty` 暴露 `.input` / `.textarea` getter，与原 `el-input` 实例接口保持一致

## 对外暴露的实例接口

与原 `el-input` 保持一致：

| 接口 | 类型 | 说明 |
|------|------|------|
| `.input` | 属性 | 底层 `<input>` DOM 元素 |
| `.textarea` | 属性 | 底层 `<textarea>` DOM 元素 |
| `.focus()` | 方法 | 聚焦 |
| `.blur()` | 方法 | 失焦 |
| `.select()` | 方法 | 全选文本 |
| `.clear()` | 方法 | 清空内容 |
| `.resizeTextarea()` | 方法 | 手动触发 textarea 高度重算 |

## 已知限制

- `type="number"` 时 `maxlength` 被浏览器忽略（HTML 规范行为，非组件问题）
- 已有 `show-word-limit` 但未传 `maxlength` 的地方，替换后会显示 `/4000` 计数器（行为变化，非 bug）
